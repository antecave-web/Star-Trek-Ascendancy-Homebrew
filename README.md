# Star Trek: Ascendancy Homebrew

A collection of unofficial variants for **Star Trek: Ascendancy** focused on improving strategic pacing, asymmetry, and player interaction while preserving the core mechanics of the original game.

Rather than rewriting the game, these projects aim to make minimal, purposeful rule changes that create new strategic experiences without sacrificing compatibility with existing expansions.

## Design Philosophy

These variants are guided by a few common principles:

* Preserve the core Ascendancy rules wherever possible.
* Favor elegant solutions over additional complexity.
* Create meaningful strategic choices through incentives rather than restrictions.
* Build on existing game systems instead of replacing them.
* Produce more cinematic game arcs while remaining familiar to experienced players.

---

## Projects

### Galactic Crisis Mode

A hybrid game mode combining the exploration and empire-building of the base game with the alliance warfare and Resistance mechanics of the Dominion War expansion.

Designed to create a more natural progression from exploration, to Cold War, to total war while reducing abrupt endings and increasing meaningful player interaction.

→ https://github.com/antecave-web/Star-Trek-Ascendancy-Galactic-Crisis-Mode

---

### Playable Borg

A complete redesign of the Borg as a fully playable asymmetric faction.

The redesign emphasizes gradual assimilation, long-term production planning, technological adaptation, and relentless expansion while remaining as compatible as possible with the official rules.

→ https://github.com/antecave-web/Star-Trek-Ascendancy-Playable-Borg
